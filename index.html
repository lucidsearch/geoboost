Exemple : Template QAPage

{
  "@context": "https://schema.org",
  "@type": "QAPage",
  "mainEntity": {
    "@type": "Question",
    "name": "VOTRE QUESTION CIBLE",
    "acceptedAnswer": {
      "@type": "Answer",
      "text": "VOTRE RÉPONSE OPTIMISÉE",
      "upvoteCount": N,
      "author": {"@type": "Organization", "name": "..."}
    }
  }
}

  2. Indexation (sitemap + ping Google)
  3. Test contre ChatGPT Search / Perplexity
  4. Mesure du taux d'extraction et de citation
  5. Itération sur le schéma et le contenu
